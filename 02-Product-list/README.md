# 🛒 TechStore

A minimal React e-commerce product catalog component structure that displays product cards with stock status, ratings, and pricing.

---

## 📁 Project Structure

```text
src/
├── components/
│   ├── ProductCard.jsx   # Individual product UI card
│   └── ProductList.jsx   # Grid container mapping over product data
├── data/
│   └── products.js       # Product dataset array
├── App.jsx               # Main layout wrapper
├── App.css               # Main styling
└── main.jsx              # Application entry point
```
