# 💡 QuizApp

A fast, interactive quiz application built with React and Vite. Test your knowledge across multiple categories with real-time scoring and instant feedback.

---

## ⚡ Features

- **Dynamic Questions:** Renders multiple-choice questions dynamically.
- **Real-Time Score:** Instant updates on correct and incorrect answers.
- **Score Summary:** End-of-quiz breakdown with percentage and restart option.
- **Timer Support:** Optional countdown timer per question.
